How to Install Your Counterstrike Custom Crosshair

Once you’re happy with the crosshair you’ve created you have two options:

Option 1: Paste crosshair code into the developer console

Copy and paste the custom crosshair code into the developer console once you load up the game. To enable counterstrike's developer console go to Options->Game Options->Enable Developer Console. Exit the options menu and press the ` (tilde) key. Paste the custom crosshair code into the console’s textarea and press enter. Enter a game to check out your new custom crosshair!

Option 2: Edit your CS:GO config.cfg file

Add the code to your counterstrike config file. Your config file will usually be located somewhere like: C:\Program Files (x86)\Steam\SteamApps\common\Counter-Strike Global Offensive\csgo\cfg if there isn’t already an config.cfg file or autoexec.cfg file: create one. Open the file and paste the custom crosshair code into the textarea. Save the file. Enter the game and check your new custom crosshair!

CS:GO Custom Crosshair Not Working?

So you've customized your crosshair just how you like it, copied the code, but it still doesn't work? Make sure you're copying the right piece of code! If you're using the developer console to install your CSGO Custom Crosshair, make sure you copy the line of code from the 'Developer Console' area. Otherwise you want to use the code in the box labeled 'config'.

Why Should I Use a Custom CS:GO Crosshair?

Some players are not satisified by the default CS:GO crosshair. It is common to customize your crosshair to give it a personalized look and feel.
